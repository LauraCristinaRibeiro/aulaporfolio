# Portfólio de Clientes

Projeto desenvolvido durante o curso **HTML e CSS: Criação de Websites**, com o objetivo de demonstrar aos alunos a construção de uma página web completa utilizando HTML5, CSS3 e JavaScript.

O sistema consiste em um modelo simples de portfólio profissional, permitindo apresentar informações pessoais, habilidades, formação, experiências e meios de contato. Durante o desenvolvimento, foram explorados conceitos fundamentais de estruturação de páginas, estilização visual, responsividade e interatividade.

---

## Objetivos de Aprendizagem

Este projeto foi desenvolvido para que os alunos pratiquem:

- Estruturação de páginas utilizando HTML5.
- Criação de layouts modernos com CSS3.
- Aplicação de Flexbox para organização dos elementos.
- Desenvolvimento de interfaces responsivas.
- Utilização de animações e transições.
- Implementação de menus responsivos.
- Manipulação do DOM com JavaScript.
- Organização de projetos para publicação no GitHub.

---

## Tecnologias Utilizadas

### HTML5

- Estrutura semântica da página.
- Navegação por âncoras.
- Organização do conteúdo em seções.
- Utilização de listas e links.

### CSS3

- Flexbox.
- Responsividade com Media Queries.
- Animações utilizando `@keyframes`.
- Efeitos de transição.
- Estilização de cards.
- Personalização de menus de navegação.
- Plano de fundo com imagem.

### JavaScript

- Seleção de elementos com `querySelector()`.
- Eventos com `addEventListener()`.
- Manipulação de classes com `classList.toggle()`.
- Implementação de menu hambúrguer para dispositivos móveis.

## Funcionalidades

### Apresentação Profissional

Área destinada à descrição pessoal ou profissional do usuário.

### Habilidades

Listagem das principais competências e conhecimentos.

### Formação e Cursos

Apresentação de cursos e formações em formato de cartões (cards).

### Contato

Exibição de informações para contato profissional.

### Menu Responsivo

Menu adaptável para dispositivos móveis através do botão hambúrguer.

### Animações

Animação suave de entrada dos elementos da página.

## Como Executar
Clone o repositório:

```text
git clone https://github.com/seu-usuario/portifolio-de-clientes.git
```

Acesse a pasta do projeto:
```text
cd portifolio-de-clientes
```

Abra o arquivo index.html em seu navegador.

Não é necessário instalar bibliotecas ou dependências adicionais.
